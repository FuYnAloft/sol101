# 28691: 字符串中的整数求和

http://cs101.openjudge.cn/practice/28691/

输入两个长度为3的字符串，每个串前两个字符是数字,后一个字符是字母。 求这两个串中的整数的和

**输入**

一行，两个字符串

**输出**

两个字符串中整数的和

样例输入

```
12B 34D
```

样例输出

```
46
```



```python
def main():
    import sys
    input = sys.stdin.read
    data = input().strip().split()

    num1 = int(data[0][:2])
    num2 = int(data[1][:2])

    result = num1 + num2
    print(result)

if __name__ == "__main__":
    main()
```
